Web API Documentation

This documentation provides detailed information about the endpoints available in our Web API.

Drivers Controller

Endpoints:

  • GET /drivers - Retrieves all drivers.
  • GET /drivers/{id} - Retrieves a specific driver by ID.
  • POST /drivers - Adds a new driver.
  • PUT /drivers/{id} - Updates an existing driver.
  • DELETE /drivers/{id} - Deletes a driver.

Upload Image Controller

Endpoints:

  • POST /uploadimage - Saves an uploaded image file.
