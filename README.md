### Prerequisites

### Installation

1. Clone the repository

```bash
git clone [https://github.com/oskarfritzner/Eksamen-Webutvikling.git](https://github.com/oskarfritzner/Fullstack-F1-App__Eksamen-Webutvikling.git)
```

```bash
cd backend
dotnet restore
dotnet run
```
